# invoices for software

Create pdf invoices for software.

![invoice demo screenshot](screenshot.png)

## features

- pdf / html invoice
- customize to your needs
- localizable

## demo

- git clone the project
- `composer install`
- `php index.php`
- check `invoice.pdf` / `invoice.html`
